SOBRE O PROJETO


Nós do curso de Sistemas para Internet da FMU, fomos desafiados a criar site usando HTML, CSS, Java Script, e Bootstrap.
Fiz esse pequeno site de Operções Matemáticas, são por enquanto operações básicas, mas tenho planos de continuar e colocar coisas mais avançadas, com o intuito de ajudar pessoas com dificuldade em matemática. Além de resolver, pretendo criar partes aonde posso ensinar essas pessoas, pode ser em vídeo ou em texto, mas trazer o conhecimento e facilitar a vida.

Usei Intellij com uma licensa de estudante para desenvolver. Utilizei HTML5, CSS3, JavaScript, e o Bootstrap 3.3.7.
Acabei utilizando uma versão mais antiga do Bootstrap pois tinha mais artigos e páginas de dicas na internet. Com a versão 4 não consegui fazer um bom menu...

O site está totalmente responsivo, foi utilizado o Bootstrap para isso. Então o menu, footer (rodapé), imagens e outros elementos se adaptam ao dispositivo, isso inclui smartphones e outros devices.
